RayPy Simple 2D Optics Simulation
===============================
author: Tobias Schoch

Overview
--------

A simple simulation for 2D arrangments of basic optical elements


Change-Log
----------

Installation / Usage
--------------------

To install use pip:

    pip install <git-url>


Or clone the repo:

    git clone <git-url>
    python setup.py install
